### 2. CAMADA DE SERVIÇO (API)
- **frontend/src/services/api/modalidadesService.ts**
  - `listarModalidadesApi(page, limit)` → GET /modalidades com paginação
  - `buscarModalidadePorIdApi(id)` → GET /modalidades/:id
  - `criarModalidadeApi(dados)` → POST /modalidades
  - `atualizarModalidadeApi(id, dados)` → PUT /modalidades/:id
  - `desativarModalidadeApi(id)` → PATCH /modalidades/:id/desativar
  - `ativarModalidadeApi(id)` → PATCH /modalidades/:id/ativar
  - Import correto de `./axios` (api configurado com interceptor de token)

### 3. REACT QUERY HOOKS
- **frontend/src/hooks/useModalidades.ts**
  - `useModalidades(page, limit)` → query com cache e refetch automático
  - `useModalidade(id)` → query por ID individual com enabled condicional
  - `useCriarModalidade()` → mutation com invalidação de cache
  - `useAtualizarModalidade()` → mutation com invalidação de cache específico
  - `useDesativarModalidade()` → mutation com invalidação de cache
  - `useAtivarModalidade()` → mutation com invalidação de cache
  - QueryClient invalidation estratégica após mutações
